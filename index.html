What is RembrandtOS?

RembrandtOS is a browser-based Linux terminal simulator built entirely in JavaScript. There's no real OS, no real network — everything is simulated, including the filesystem, running processes, and network responses from target machines.

It's designed to let you practice real penetration testing workflows — from recon all the way to domain compromise — without needing a lab environment.

⚙️
How it works
FS The filesystem is a JavaScript object. Files you create with touch or nano persist for the session.
CMD Commands are matched against a handler list — each one returns realistic simulated output.
NET The simulated network has real targets: a domain controller at 10.10.10.10, a legacy box, and more.
UX Tab completion, command history (↑↓), multi-tab terminals, and a file manager all work.
🏆
The CTF Lab

The main lab walks you through a full Kerberoasting attack chain — network recon, SMB enumeration, SPN discovery, ticket cracking, and lateral movement to Domain Admin.

A second lab covers EternalBlue / MS17-010 via Metasploit. More labs are on the way. Track your progress in the CTF panel on the right, or type help in the terminal for a step-by-step guide.
